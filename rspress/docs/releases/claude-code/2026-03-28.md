## 📌 主な変更点
- APIリクエストに`X-Claude-Code-Session-Id`ヘッダーを追加し、プロキシがセッション単位でリクエストを集約しやすくなった
- `.jj`（Jujutsu）・`.sl`（Sapling）ディレクトリをVCS除外リストに追加し、GrepやファイルオートコンプリートがVCSメタデータに潜り込まなくなった
- v2.1.85以前のセッションで`--resume`が失敗するバグを修正
- プロジェクトルート外のファイル（例：`~/.claude/CLAUDE.md`）でWrite/Edit/Readが失敗するバグを修正
- スキル呼び出しのたびに不要な設定ファイル書き込みが発生し、Windowsで設定破損を引き起こす可能性があった問題を修正
- Readツールがコンパクトな行番号形式を採用し、変更のない再読み込みを重複排除することでトークン使用量を削減
