## 主な変更点
- プラグイン依存関係の強制: 依存関係があるプラグインは無効化できず、有効化時はトランジティブ依存関係も自動で有効化される
- `/plugin` マーケットプレイスにコンテキストコストの見積もり（ターンごと・呼び出しごとのトークン数）を表示
- `worktree.bgIsolation: "none"` 設定を追加し、ワークツリーが使えないリポジトリでバックグラウンドセッションが作業コピーを直接編集可能に
- PowerShellツールがデフォルトで `-ExecutionPolicy Bypass` を渡すようになった（`CLAUDE_CODE_POWERSHELL_RESPECT_EXECUTION_POLICY=1` で無効化可能）
- バックグラウンドセッションがアイドルから復帰後もモデルとエフォートレベルの設定を保持するように修正
- `.credentials.json` の破損や `stop` フック、`/loop` のキャンセルなど多数のバグを修正
