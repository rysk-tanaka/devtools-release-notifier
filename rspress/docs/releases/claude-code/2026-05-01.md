## 主な変更点
- `claude project purge` コマンドを追加。プロジェクトのトランスクリプト・タスク・ファイル履歴・設定をまとめて削除可能（`--dry-run` / `--all` オプション対応）
- `/model` ピッカーが `ANTHROPIC_BASE_URL` 経由のゲートウェイにある `/v1/models` エンドポイントからモデル一覧を取得するように改善
- WSL2・SSH・コンテナ環境など、ブラウザコールバックが localhost に到達できない場合に `claude auth login` でターミナルへの OAuth コード貼り付けによる認証が可能に
- セキュリティ修正: 上位の管理設定に `sandbox` ブロックがない場合に `allowManagedDomainsOnly` / `allowManagedReadPathsOnly` が無視される脆弱性を修正
- Windows で日本語・韓国語・中国語が文字化けするバグ、2000px 超の画像貼り付けでセッションが壊れるバグなど多数のバグを修正
