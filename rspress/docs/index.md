## 通知について

このシステムは、各ツールの新しいリリースを自動的に検出し、日本語に翻訳して通知します。

- GitHub Releases
- Homebrew Cask
- GitHub Commits

上記のソースから優先度順に情報を取得し、新しいバージョンが見つかった場合に通知します。
